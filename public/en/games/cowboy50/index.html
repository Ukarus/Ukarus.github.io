Cowboy 50

Cowboy 50 is a game i made as a final project for the CS50 course. It’s a basic shoot’em up that includes powerups, spawning enemies, sounds, music and a basic gameplay. It’s a small clone of the mini game Journey of the Prairie King which can be played in Stardew Valley.

Tools used.
